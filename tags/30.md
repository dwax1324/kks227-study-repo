# 오일러 경로(Eulerian Path), 오일러 회로(Eulerian Circuit)


### 이론

```

무향이나 유향 그래프가 있을 때, 
그래프에 존재하는 모든 간선을 정확히 한번씩만 방문하는 연속된 경로가
오일러 (경로/회로)이다.

시작점과 도착점이 같으면 오일러 회로(eulerian circuit)이 되고,
사직잠과 도착점이 다르면 오일러 경로(eulerian path)가 된다.

무향 그래프의 경우, 차수(degree)가 홀수인 정점이 2개일때 오일러 경로, 0개일 때 오일로 회로가 존재한다.

오일러 경로는 시작점과 끝점을 차수가 홀수인 정점 2개로 하고,
오일러 회로는 존재만 한다면 그 어떤 정점도 시작점으로 가능하다.

들어오는 간선이 있다면 나가는 간선도 있기 때문에 항상 차수가 2의 배수꼴로 붙게된다.

유향 그래프에서 오일러 회로가 존재하려면 모든 정점에 대해 indegree와 outdegree가 일치해야한다.
회로가 아닌 오일러 경로라면 indegree가 1 적은 정점 하나가 시작점,
out degree가 하나 적은 정점이 끝점으로 정해지며
다른 indegree와 outdegree가 불일치하는 정점이 더 있으면 안된다.

유향/무향 관계없이, 서로 다른 컴포넌트에 각각 간선이 존재해도 오일러 경로는 존재할 수 없다.
두 컴포넌트 자체가 떨어져있어서 경로를 이을 수 없기 때문이다.


오일러 회로를 구하는 알고리즘👇
Hierholzer's Algorithm(히어홀저 알고리즘)
1. 아무정점 v를 뽑고 v에서 출발해 v로 돌아오는 경로를 하나 뽑는다.
2. 이 때,위 경로에 속해있는 정점 중 인접한 간선들 중에 경로에 쓰이지 않는, 즉 아직 방문되지못한 간서이 있는 정점u가 존재하면, 
u에서 시작해서 아직 쓰이지 않은 간선들만 사용해 u로 돌아오는 경로를 하나 더 찾아 원래의 경로에 끼워넣는다.

방문을 하면서 해당 간선을 사용한 것으로 표시하거나 지우고 간선 양쪽의 차수를 1씩 줄여나가고,
해당 정점의 방문함수가 끝나는 순간에 정점 번호를 출력하면
이걸 이어 붙인게 정답이된다.


A -- D
|    |
B -- C --E
      \ /
       F

이런 그래프가 있다고 하자.
DFS로 재귀적으로 해당 정점의 방문 함수가 끝나는 순간에 정점 번호를 출력한다.  
예를 들어 A->B->C 순으로 방문했다고 하자. 여기서 D를 선택하면,
D->A로가서 A의 인접한 간선이 모두 방문됐으므로 다시 돌아오면서
A,D를 출력한다.
그 다음 남은 정점 E를 선택해서 C->E->F->C로 가면 C도 더이상
이웃이 없어서 그대로 돌아가며 C,F,E,C를 출력한다.
C에서도 시작점인 A로 돌아가면서 B,A를 출력하게되고,
결국 "A D C F E C B A"가 되고 오일러회로가 된다.



```
